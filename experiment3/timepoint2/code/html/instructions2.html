To illustrate what we mean, consider the following vignette.

You have just been told that rumours have been going around about you on social media. Which of the following situations would you rather be in?
A. The rumours are true and positive.
B. The rumours are false and negative.

In this case, you'd want to answer "definitely A" or "probably A" because that is a much nicer situation to be in.
